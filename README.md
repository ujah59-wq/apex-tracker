# ⚡ APEX Tracker v2

## Quick Start

```bash
npm install
npm run dev
```

Opens at **http://localhost:3000**

On your phone, use the **Network** URL shown in the terminal (e.g. `http://192.168.x.x:3000`)
Make sure your phone is on the same WiFi as your computer.

## Find your IP
- Mac: `ipconfig getifaddr en0`
- Windows: `ipconfig` → IPv4 Address
- Linux: `ip addr show`
